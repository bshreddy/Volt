# Vortex-Optimized Lightweight Toolchain (VOLT)
Volt is an end-to-end, open-source compiler framework for the Vortex open-source GPU and its derivatives. It supports multiple programming models, including CUDA and OpenCL, by extending two open-source projects: PoCL and CuPBoP. Volt also extends LLVM to perform SIMT-aware, IR-level optimizations for portability across Vortex GPU variants, and augments the LLVM RISC-V backend to provide full support for the Vortex ISA.

## 1. Front-end Compiler
The front-end compiler compiles both host and kernel code. For kernel compilation, the front-end analyzes the semantics of the source language and emits LLVM IR with semantic-aware optimizations and generated thread-scheduling logic. For host compilation, CuPBoP-Vortex and PoCL-Vortex employ a runtime translation layer and link against the host runtime library to interface with the Vortex device driver and runtime stack.

Currently, Volt supports two languages: CUDA and OpenCL.

- [PoCL-Vortex](https://github.com/vortexgpgpu/pocl): Front-end compiler for OpenCL support. See the [PoCL-Vortex docs](./3.PoCL_vortex.md) for details.
  - OpenCL Version: 3.0
- [CuPBoP-Vortex](https://github.com/cupbop/CuPBoP_Vortex): Front-end compiler for CUDA support. See the [CuPBoP-Vortex docs](./4.CuPBoP_vortex.md) for details.
  - CUDA Version: 12.0

## 2. Middle-end Compiler
The middle-end compiler applies target-independent optimizations and GPU-aware analyses, enabling reuse across GPU architectures. One of the most critical challenges in GPU compiler design is providing effective support for divergence control, analysis, and performance optimization. 

Volt extends LLVM to implement the middle-end compiler. See the [LLVM-vortex docs](./6.LLVM_vortex.md) for details.

- [LLVM-Vortex](https://github.com/vortexgpgpu/llvm): Middle-end compiler for centeralized optimizations
  - Based on Version 18.0

## 3. Back-end Compiler
The back-end compiler performs target-specific optimizations and final code generation. Volt extends the LLVM RISC-V target backend to generate target binaries for the Vortex GPU.
- [LLVM RISC-V target backend](https://github.com/vortexgpgpu/llvm)
